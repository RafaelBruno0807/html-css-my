Usando links externos

Você pode acessar o meu repositório público no Github

Para criar um acesso a outro site através de um link você pode usar a tag A que significa âncora e envelopar o texto que você quer usar para inserir o link.
Dentro da tag A tem um parâmetro chamado de HREF que significa hypertext reference(referencia de hypertexto) e dentro desse parâmetro você pode inserir o link para onde a pessoa que clicar vai ser redirecionada.
Dentro da tag A após inserir o HREF com o link você pode usar também o parâmetro TARGET que significa alvo e colocar dentro de target _BLANK para que quando a pessoa clique no link o navegador vai abrir outra aba para encaminha-lá.
Após usar a parâmetro TARGET você precisa usar o parâmetro REL que significa "relativo à" e aí você fala se é um link interno ou externo. Como eu usei o meu Github que é um link externo eu vou colocar EXTERNAL dentro do parâmetro REL mas se fosse um link interno eu colocaria INTERNAL.
